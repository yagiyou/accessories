Brand / 品牌介紹

我們相信「飾品不是為了耀眼，而是為了陪伴」。
Komorebi（木漏れ日）象徵樹葉間灑落的光——溫柔、細碎、卻讓人安心。
每件作品以可日常配戴為前提，保留材質的原始質感，讓光澤在動作之間慢慢發亮。

Materials / 材質與工法

・材質：925銀 / 黃銅 / 天然石 / 珍珠
・工法：手工打磨、拋光、簡約線條成形
・服務：尺寸微調、客製刻字（可選）、包裝禮盒

Collection / 作品集

點擊作品可查看：系列概念、材質、製作細節。

Notes / 日系手作的細節

・色調：柔和自然光、保留金屬與石紋的真實質感
・風格：留白、乾淨、細節更靠近生活
・拍攝：建議用窗邊自然光，讓光澤「慢慢」出來
